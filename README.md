# Otimizacao-da-Topologia-do-Arranjo-Submarino
Projeto de Otimização da Topologia do Arranjo Submarino de um campo produtor de petróleo. 

Disciplina: Matematica Computacional - 2026.1

O projeto consiste em: 

* Localização Otimizada: A determinação exata das coordenadas geográficas (X, Y) para a instalação do manifold que garante o menor custo global de interligação.

* Redução de Custos (CAPEX): A minimização matemática do comprimento total dos dutos e umbilicais, traduzindo-se em uma economia direta e substancial de materiais para o projeto offshore.

* Alta Fidelidade Física: A comprovação de que o uso do relevo 3D (batimetria via splines) evita o subdimensionamento perigoso que ocorreria caso o cálculo fosse feito em um plano 2D simplificado.

* Validação Computacional: A demonstração prática de que os métodos clássicos de Cálculo Numérico (Descida de Gradiente, Integração e Diferenças Finitas) são perfeitamente capazes de resolver problemas complexos e não-lineares da engenharia moderna de forma estável e precisa.
